Git Intro

Back to Top

What is Git? Software Version Control.

In software development there is often multiple authors working on the same documents. Git is used to track the changes made to projects and record a complete history.

Git Download Page »

But Why? Collaboration!

Git makes it easy to share your projects by allowing other users to clone them and make their own version.

But you're more of a solo ninja. Epic.

Git excels at backing up your own projects. Once you push projects to github.com you no longer have to worry about losing your precious scripts again.

Clone it

Open your Command Prompt, navigate to your project folder using a combination of: "c:", "dir", "cd foldername" and then type "git clone url"

Check Project Status

Check the project status with "git status". Update to the latest revision with "git checkout".

Add Changes

Prepare adding a change with "git add filename" or "git add ." to add all files in the current directory.

Commit

Use git commit -m "message" to notate your commit and "git push" to upload new code to the project.
